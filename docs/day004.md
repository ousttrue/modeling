## Script を AddOn 化

AddOn 化して定形作業をボタン一発で動くように構築。

- Object を名前ベースで検索して Humanoid に割当
- Object から Link を生成
- Link を Armature の目的のボーンに親子化
- ボーンの長さでスケール

これを Assemble 機能とする。

逆に既存の Skinning Model を部品に分解する、 Disassemble 機能が作れそうなのでやってみよう。
分解してからモデルを観察することで発見がありそうな気がする。
